# POE 过滤器同步编辑器 (POE Filter Comparator)

一款专为《流放之路》(Path of Exile) 玩家和过滤器制作者设计的可视化比对与编辑工具。它可以帮助你轻松同步不同版本过滤器之间的差异，尤其是用于精准更新 BaseType 和处理自定义过滤逻辑。

![Version](https://img.shields.io/badge/version-v11.0-green)
![Python](https://img.shields.io/badge/Python-3.x-blue)

## 核心功能

- **双向比对**：同时加载两个过滤器文件，自动识别并提取彼此多出的代码块。
- **精准定位**：在比对结果区点击任意代码块，编辑器会自动在原文件中高亮并定位到对应位置。
- **BaseType 快速替换**：一键将比对文件中的 `BaseType` 行同步到主文件中，省去手动复制粘贴的烦恼。
- **可视化多栏布局**：采用四栏布局（主原文件、主多出内容、比对多出内容、比对原文件），逻辑清晰。
- **动态拖拽**：支持通过拖拽手柄调整各栏位的顺序，适应不同的操作习惯。
- **内置搜索**：支持在各个栏位进行实时搜索，并自动关联代码块定位。

## 使用说明

1. **加载文件**：
   - 点击“主文件”按钮加载你的常用过滤器。
   - 点击“比对文件”按钮加载参考过滤器（如官方更新版或其他作者的版本）。
2. **执行比对**：
   - 点击“开始比对”，中间两栏将显示两个文件之间互不匹配的代码块。
3. **同步与编辑**：
   - 点击结果栏中的代码块，原文件区将同步跳转。
   - 使用“⚡ 替换 BaseType”功能快速同步选中的物品类别。
4. **保存**：
   - 编辑完成后，点击对应栏位的“💾 保存”按钮即可直接写入磁盘。

## 运行与发布

- **源码运行**：需要安装 Python 及 Tkinter 环境，运行 `python filter_comparator.py`。
- **可执行文件**：在 [Releases](../../releases) 页面可以下载预编译的 `filter_comparator_v11.exe`，无需安装环境直接运行。
